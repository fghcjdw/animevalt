  ANIMEVAULT — CHROMEBOOK INSTALL GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1 — HOST THE FILES
  The app needs to be accessible via a URL.
  Options (all free):
  
  A) GitHub Pages (recommended):
     1. Create a free account at github.com
     2. Create a new repository (e.g. "animevault")
     3. Upload index.html, manifest.json, sw.js
     4. Go to Settings → Pages → Deploy from main branch
     5. Your URL: https://yourusername.github.io/animevault

  B) Netlify Drop (easiest):
     1. Go to netlify.com/drop
     2. Drag and drop the entire animevault folder
     3. Get an instant URL like https://random-name.netlify.app

  C) Vercel:
     1. Go to vercel.com
     2. Import or drag your folder
     3. Deploy for free

STEP 2 — INSTALL ON CHROMEBOOK
  1. Open Chrome on your Chromebook
  2. Go to your hosted URL (from Step 1)
  3. Click the install icon (⊕) in the address bar
     OR go to: Chrome menu (⋮) → "Install AnimeVault..."
  4. Click "Install"
  5. The app now appears in your Chromebook's app launcher!
     It works like a real installed app.

STEP 3 — USING THE APP
  • Browse anime from the home screen
  • Use the tabs: Trending, Top Rated, Airing Now
  • Filter by genre using the left sidebar
  • Search for any anime by name
  • Click a show → select an episode → it plays inside the app!

  NOTE ABOUT VIDEO PLAYBACK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  The app embeds video from aniwatch.to inside the player.
  Some sites block embedding — if a video doesn't load, 
  the app will still show you what's available and you can
  try alternative sources.

  The anime information (titles, covers, ratings) comes from
  the free Jikan API (MyAnimeList data) — no account needed.
